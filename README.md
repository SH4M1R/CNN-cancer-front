Skin Cancer - CNN
Aplicativo web para el sistema de análisis y detección de cáncer de piel mediante inteligencia artificial. Construido con React, Vite y Tailwind CSS.
Instalación

Clonar el repositorio:
git clone https://github.com/tu-usuario/cnn-front.git
cd cnn-front

Instalar dependencias:
npm install

Ejecutar proyecto:
npm run dev

Backend del proyecto:
https://github.com/SH4M1R/CNN-cancer-back.git
